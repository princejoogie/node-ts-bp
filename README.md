# node-ts-bp

Node.js Typescript Boilerplate

## Scripts

```bash
pnpm dev # runs tsx in watch mode
```

```bash
pnpm build # runs tsup and produce output in `dist/` folder
```

```bash
pnpm start # runs `index.js` from `dist/` folder
```

```bash
pnpm format # runs prettier in `src/` folder
```
